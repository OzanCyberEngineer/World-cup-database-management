# World Cup Database Management System

A relational database project designed to store and manage World Cup data, including teams, players, matches, stadiums, tournaments, goals, cards, and match statistics.

## Project Overview

This project was developed as part of a Database Management course. It demonstrates relational database design, table relationships, primary and foreign keys, data insertion, and SQL queries using World Cup data.

## Features

- Stores team and player information
- Manages World Cup tournaments and stadiums
- Records match results and statistics
- Tracks goals, cards, and player participation
- Uses primary and foreign key relationships
- Includes sample data
- Includes SQL queries for database analysis

## Project Files

| File | Description |
|---|---|
| `01_schema_base.sql` | Creates the main database tables |
| `02_schema_matches.sql` | Creates match-related tables and relationships |
| `03_insert_base_data.sql` | Inserts general sample data |
| `04_insert_match_data.sql` | Inserts match and tournament data |
| `05_queries.sql` | Contains SQL queries used to retrieve and analyze data |
| `Project Report.docx` | Detailed project documentation and explanation |
